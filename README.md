# ICETask1-PROG5121

This is our first ICE Task for PROG5121 and this application was written in Netbean using JAVA.

## Description
This is a simple Java program that asks the user to enter two numbers and prints all numbers between them.  
- If the first number is smaller, it prints in **ascending order**.  
- If the first number is larger, it prints in **descending order**.  
- If the numbers are equal, it shows an **invalid input message**.

## Example

Enter first number: 3
Enter second number: 7

3
4
5
6
7
